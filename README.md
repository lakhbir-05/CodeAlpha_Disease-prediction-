# 🏥 HealthAI Diagnostic System

An AI-powered healthcare web application to predict the likelihood of three major diseases:

- 🩺 Diabetes
- ❤️ Heart Disease
- 🎗️ Breast Cancer

The application provides an interactive user interface, secure login, disease risk prediction, and downloadable medical reports in PDF format.

---

# 📌 Features

- 🔐 Login Authentication
- 🩺 Diabetes Prediction
- ❤️ Heart Disease Prediction
- 🎗️ Breast Cancer Prediction
- 📄 PDF Medical Report Generation
- 📈 Risk Score Visualization


# 🩺 Disease Prediction Modules

## 1. Diabetes Prediction

Input Parameters

- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

Output

- Low Risk / High Risk
- Probability Score
- PDF Report

---

## 2. Heart Disease Prediction

Input Parameters

- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels
- Thalassemia

Output

- Low Risk / High Risk
- Probability Score
- PDF Report

---

## 3. Breast Cancer Prediction

Input Parameters

- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Concave Points
- Symmetry
- Fractal Dimension

Output

- Benign
- Malignant
- PDF Report
